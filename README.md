# So_long-2D-game
Small 2d game \
To play: \
1.) clone repository. \
2.) type make for the simple implementation of the game or make bonus for animated game. \
3.) launch the program with a map as a parameter: a file with .ber extension. \
Some examples of maps are in maps/ folder(also invalid maps for error testing). \
4.) to exit: press x in the corner, ESC or complete the game by collecting all collectibles(crystals) and step on the exit.
